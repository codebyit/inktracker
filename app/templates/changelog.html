v0.9.0 – Multi-Craft Support

Released 2026-06-28

New Features

  • Multiple craft modes per project – Define different finishes for each face (e.g., 2-sided case: Side A = Flat Raised, Side B = Flat)
  • Craft variant breakdown in project detail view
  • Enhanced wizard UX for multi-face projects

Technical Updates

  • Tailwind CSS v4 for faster builds and improved styling
  • Multi-craft data persistence in SQLAlchemy models

Security

  • Hardened the project print worksheet against cross-site scripting (XSS) by escaping all user- and config-derived values
  • Removed a file-access race condition in the release tooling

Compatibility

  • All existing single-craft projects work unchanged
  • Multi-craft is opt-in via project toggle in the wizard

v0.6.8 – Project Metadata Persistence

Released 2026-06-15

Improvements

  • Project metadata (material, craft mode) now persisted across edits
  • Craft ink mode synced into active ink-mode calculations
  • SQLite migration compatibility fixes

v0.6.7 – Security & Craft Mode Enhancements

Released 2026-06-01

Improvements

  • Expanded craft mode controls (Flat Raised, Pattern Texture, Relief Texture, Customize)
  • GitHub Actions security pipeline (Bandit + pip-audit)
  • Startup reliability diagnostics and migration timeout guards
